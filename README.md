Hintro-FE
A clean, responsive Kanban board built to help teams manage tasks with zero friction. I focused on making the drag-and-drop experience feel snappy and ensuring the state remains consistent across the app.

Setup & Installation
To get the project running locally, follow these quick steps:

1. Clone & Install
Bash
git clone <repository-url>
npm install
2. Spin up Development
Bash
npm run dev
Navigate to http://localhost:5173 to see it in action.
